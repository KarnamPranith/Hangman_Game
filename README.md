🎮 Hangman Game

Overview

This is a simple text-based Hangman game developed in Python as part of my CodeAlpha Python Internship.

The player guesses a hidden word one letter at a time. The game allows up to 6 incorrect guesses before the player loses.

Features

- Random word selection
- Console-based gameplay
- 6 incorrect guesses allowed
- Win/Lose conditions
- User-friendly interface

Technologies Used

- Python
- Random Module

Concepts Used

- Variables
- Strings
- Lists
- Loops
- Conditional Statements
- User Input

How to Run

1. Download the repository.
2. Open terminal in the project folder.
3. Run:

python hangman.py
